# Minhas Receitas

## Objetivo
- Construir uma Landing Page, trazendo meus novos conhecimentos adquiridos com o tempo de estudo, esforço e prática.
- E com React Rounter ter páginas, internas e aninhadas para:
    - Pães.
    - Salgados.
    - Molhos.
    - Bolos.
    - Doces.
    - Recheios.

## Tecnologias Utilizadas
### Linguagens
- HTML
- CSS
- JavaScript
### Bibliotecas
- React + Vite
- React Rounter
